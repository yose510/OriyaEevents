## 🚀 התקנה מהירה (15 דקות)

### שלב 1: הגדרת Database ב-Supabase (5 דקות)

1. **עבור ל-Supabase SQL Editor:**
   ```
   https://supabase.com/dashboard/project/kzxyewemuykimsblotif/sql/new
   ```

2. **העתק את כל התוכן מ-`database-schema.sql`**

3. **לחץ RUN (Ctrl+Enter)**
   - ✅ צריכות להיווצר 5 טבלאות:
     - users
     - packages (עם 10 חבילות מוכנות!)
     - invitations
     - invitations_sent
     - payments

4. **כבה אימות מייל (חשוב!):**
   - Settings → Auth → Email → **כבה** "Enable email confirmations"
   - שמור!

5. **הגדר URL Redirects:**
   - Settings → Auth → URL Configuration
   - **Site URL:** `https://yose510.github.io/OriyaEvents/`
   - **Redirect URLs:** `https://yose510.github.io/OriyaEvents/**`
   - שמור!

### שלב 3: יצירת משתמש אדמין (5 דקות)

#### אופציה 1: הרשמה רגילה והפיכה לאדמין

1. **עבור לאתר:**
   ```
   https://yose510.github.io/OriyaEvents/
   ```

2. **לחץ "הרשם עכשיו"**

3. **מלא פרטים:**
   - שם: יוסף כהן (או כל שם)
   - אימייל: admin@oriya.com (או כל מייל)
   - סיסמה: מה שתרצה (לפחות 6 תווים)

4. **לחץ "הרשם"** → יעביר אותך ישר ל-dashboard

5. **הפוך לאדמין ב-Supabase:**
   - Supabase → Table Editor → `users`
   - מצא את המשתמש שיצרת
   - שנה `is_admin` מ-`false` ל-`true`
   - שמור

6. **רענן את הדף** → יעביר אותך ל-admin panel! 🎉

#### אופציה 2: יצירה ישירה ב-Supabase

1. **Supabase → Authentication → Users → Add User**
2. **מלא:**
   - Email: admin@oriya.com
   - Password: הסיסמה שלך
3. **Supabase → Table Editor → users → Insert Row:**
   ```
   id: (העתק את ה-UUID מ-Authentication)
   email: admin@oriya.com
   full_name: מנהל המערכת
   package_invites: 0
   package_sent: 0
   package_paid: false
   is_admin: true  ← חשוב!
   ```

## 🗄️ מבנה Database

```sql
users (משתמשים)
├── id (UUID)
├── email
├── full_name
├── package_invites (כמה בחבילה)
├── package_sent (כמה נשלחו)
├── package_paid (שולם?)
└── is_admin (מנהל?)

packages (חבילות מוכנות)
├── id
├── name ("חבילת 300 - חתונה")
├── invites (300)
├── price (300.00)
├── event_type ("wedding")
└── is_active

invitations (הזמנות)
├── id
├── user_id
├── event_type (wedding/bar_mitzvah/bat_mitzvah)
├── title
├── groom_name / bride_name / bar_bat_name
├── event_date
├── event_time
├── venue
├── address
├── template (classic/modern/elegant/floral)
├── background_color
└── custom_text

invitations_sent (מעקב)
├── id
├── user_id
├── invitation_id
├── guest_name
├── guest_phone
└── sent_at

payments (תשלומים - עתידי)
├── id
├── user_id
├── package_id
├── amount
├── status
└── payment_proof
```
